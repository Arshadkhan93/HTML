History of HTML

HTML (HyperText Markup Language) evolved from Tim Berners-Lee's initial concept of a hypertext system for sharing documents at CERN in the late 1980s. The first public description of HTML appeared in 1991, and the first standardized version, HTML 2.0, was released in 1995, laying the foundation for web development. Subsequent versions, including HTML 3.2, 4.0, and XHTML, introduced new features and stricter syntax, culminating in the widespread adoption of HTML5 as the standard for web development.
